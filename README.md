# Casio PB-1000 C Cross Compiler (WIP)

**Status:** Under construction. This repository is currently documentation-first and will evolve into a working compiler implementation.

## What this is
A C cross-compiler project targeting the **Casio PB-1000** (Hitachi **HD61700** CPU), designed around real constraints of the PB-1000 assembler and validated via repeatable test harnesses.
